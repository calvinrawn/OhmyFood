# Ohmyfood - Dynamisez une maquette avec des animations CSS

Troisième projet de ma formation, **Ohmyfood** est un site "mobile-first" qui permet de composer son propre menu et de réduire le temps d'attente au restaurant. Ce projet m'a permis d'approfondir l'utilisation de Sass et de créer des animations fluides sans JavaScript.

## 📝 Objectifs du projet
* Intégrer une maquette complète en mode **Mobile-first**.
* Utiliser **Sass** pour structurer le code CSS (variables, mixins, nesting).
* Implémenter des animations CSS avancées pour améliorer l'expérience utilisateur.

## 🚀 Compétences techniques
* **Sass (SCSS)** : Organisation du code via l'architecture 7-1 (partials), utilisation de variables pour la charte graphique.
* **Animations CSS** : 
    * Création d'un **loading spinner** au chargement de la page.
    * Apparition progressive des plats du menu (décalage temporel).
    * Animation des boutons et du système de sélection "Check".
* **Responsive Design** : Adaptation parfaite du mobile au desktop via les Media Queries.

## 🛠️ Outils utilisés
* **HTML5 / CSS3 / Sass**
* **Visual Studio Code**
* **Git / GitHub** (Gestion de version)
* **FontAwesome**
